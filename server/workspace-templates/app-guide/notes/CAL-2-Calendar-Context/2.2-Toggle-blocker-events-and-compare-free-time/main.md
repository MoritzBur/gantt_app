# Toggle blocker events and compare free time

![Sailor calendar scenario](../_images/sailor-scenario.png)

This screenshot uses the sailor demo because it shows the idea cleanly:

- the red-shaded coaching weekends are fixed commitments that are already active
- the orange regatta options are visible but still optional
- the thesis bars below show what gets squeezed when you start committing more time

Try this:

- Double-click one optional calendar event to activate it as a blocker.
- Look at the overlapping tasks and their available workday count.
- Activate a second event and compare again.

This is great for scenario planning:

- "If I say yes to this trip, what gets squeezed?"
- "Where is the next real deep-work block?"
