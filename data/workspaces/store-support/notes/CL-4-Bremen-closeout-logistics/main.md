# Bremen closeout logistics

The store is already closed, but the project is still operationally messy.

- Fixtures split into resale, transfer, and disposal lots.
- Archived paperwork has to be scanned before handover.
- The landlord inspection is strict on patching walls and emptying the cellar.

Use this project to show how one person can juggle travel, warehouse dependencies, and vehicle blockers at the same time.
