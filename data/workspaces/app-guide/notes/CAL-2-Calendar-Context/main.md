# Calendar context

The main idea of the app is not just planning work in the abstract.

![Header with calendar status](./_images/header.png)

The calendar controls live in the top-right header:

- `Calendar connected` tells you whether this workspace currently has calendar context.
- `Manage` opens calendar setup.

This helps you ask:

- What time is really left once my actual calendar is taken into account?
- Which plan still works if I commit to specific events?

This guide workspace ships with a tiny example personal calendar so you can try the interaction immediately.
