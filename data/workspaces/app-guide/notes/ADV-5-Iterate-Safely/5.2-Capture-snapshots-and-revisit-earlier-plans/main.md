# Capture snapshots and revisit earlier plans

![Header with history button](../_images/header.png)

Use the top header for this feature.

- `History` opens the snapshot panel.
- `● Uncommitted changes` tells you that the current plan differs from the last saved snapshot.
- the `↶` and `↷` buttons in the toolbar handle quick undo and redo

Why this matters:

- you can try a risky planning idea without fear
- you can name important plan versions
- you can revisit an earlier state when a conversation changes direction

Suggested experiment:

- make one visible change to a task bar
- try `Ctrl/Cmd + Z` and `Ctrl/Cmd + Shift + Z` or `Ctrl/Cmd + Y`
- open `History`
- create a snapshot with a short message like `Option A before client review`
- use `Ctrl/Cmd + Enter` inside the snapshot panel if you want to save it quickly
- keep iterating, knowing you can come back
