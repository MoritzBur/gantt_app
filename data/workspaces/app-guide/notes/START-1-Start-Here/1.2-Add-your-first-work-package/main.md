# Add your first work package

Stay in the left task tree for this step.

- Right-click a group row if you want to add a sibling or child item nearby.
- Keep an eye on the bars on the right so the new work lands in a sensible place.

Suggested experiment:

- Create one group for a real project area.
- Add one or two tasks underneath it.
- Move the bars until the shape feels roughly right.

Useful gestures here:

- right-click a row to add nearby items
- drag the handle on the left to reorder rows
- hold `Shift` while dragging a group bar in the timeline to move the whole group

This is the moment the app stops being a demo and starts becoming your own planning space.
