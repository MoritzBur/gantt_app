# Rename tasks and drag bars

![Planning area](../_images/workspace-overview.png)

Use the center of the screen for this first exercise:

- Click this task in the left task tree to keep it selected.
- Use the task row on the left if you want to rename or edit it.
- Use the matching blue bar on the right if you want to move it in time.

Try this now:

- Rename this task to something from your real life.
- Drag the bar left or right.
- Stretch the bar longer or shorter.
- Double-click the row if you want to open the task editor directly.

Why this is cool:

- You are not filling in a rigid setup form first.
- You can sketch a plan in seconds.
- The schedule becomes useful before it becomes perfect.

Small interaction tips:

- single click previews the note
- double-click opens the editor
- shift + double-click opens the note as a pinned tab
